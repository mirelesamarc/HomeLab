# HomeLab
This repository documents my hands-on lab for building a virtualized Windows Server 2019 environment with a Windows 10 client. I built this to strengthen my understanding of Active Directory, DHCP, and networking fundamentals.
# Features
Windows Server 2019 Installation
Active Directory (AD DS) Setup
DHCP Server Configuration
Network Address Translation (NAT) for Internet Access
Windows 10 Domain Join
# Requirements
- [Oracle VirtualBox](https://www.virtualbox.org/)
- [Windows Server 2019 ISO](https://www.microsoft.com/en-us/evalcenter/evaluate-windows-server)
- [Windows 10 ISO](https://www.microsoft.com/en-us/software-download/windows10)
